## Project Overview

### Problem Statement
Traders and companies cannot efficiently anticipate price fluctuations for energy and metals commodities.

### Solution
An AI-powered platform that collects, processes, and analyzes commodity data to provide actionable price intelligence.

---

## Platform Features

### Data Collection
- Historical commodity prices
- News and geopolitical events

### AI Capabilities
- **Time-series prediction** for price forecasting
- **Sentiment analysis** on news articles
- **LLM-powered interpretation** ("Why did the price spike?")

### Technical Components
- API data extraction
- Feature engineering pipeline
- Data ingestion system
- LLM integration
- Dashboard for visualization

## Project Requirements

### Mandatory Requirements
- [ ] AI component (required)
- [ ] Data flow pipeline: collection → processing → output
- [ ] Functional MVP
- [ ] Real/justified use-case
- [ ] Git repository with source code

### Suggestions (Appreciated)
- Financial/energy/health APIs
- Time-series data handling
- Data pipelines (batch or streaming)

### Not Accepted
- Generic chatbot without clear purpose
- Applications without real data component
- Shallow AI usage ("just added a prompt")
- Projects without functional demo
